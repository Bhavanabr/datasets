# multitool

Swiss army knife to work with the dataset. Written in Go, easy to install and use.

The following commands exist:

* `discover`
* `select`
* `get-index`
* `get-dataset`

## Installation

```
go get github.com/src-d/datasets/sourset/multitool
```
